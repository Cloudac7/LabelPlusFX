# 新建翻译

 - 流程
   - 判断是否确认新建（是否留在当前状态）
   - 显示文件选择器
     - 默认文件名称为“Nova traduko”
     - 默认文件扩展名为JSON或者TXT
     - 被关闭了，流程中止
     - 决定新建一个文件
       - 如果最终输入的文件名
         - 文件名
           - 是默认文件名：替换为当前文件夹的名称
           - 不是默认文件名：使用输入的文件名
         - 扩展名
           - 不存在：使用默认扩展名
           - 不是JSON或者TXT：使用默认扩展名
           - 是JSON或者TXT：使用输入的扩展名
       - 继续 
   - 如果文件已经存在，询问是否覆盖（显示冲突的文件名称）
     - 不覆盖：流程中止
     - 覆盖：继续
   - 重置状态
   - 执行新建操作
   - 得到新建操作的返回值，即项目文件夹
     - 为null：新建失败，流程中止
     - 为File：新建成功，执行打开
